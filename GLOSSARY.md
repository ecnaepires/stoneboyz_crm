# Coding With This CRM Glossary

This glossary tracks coding terms only after they have been taught and used correctly during code-along sessions.

## Terms

**Variable**:
A named value the code can reuse, such as `squareFeet`, `rate`, or `finalPrice`.
_Avoid_: Box, thing

**Expression**:
Code that produces a value, such as `squareFeet * rate`.
_Avoid_: Formula, math part

**Object**:
A value made of named fields, such as a price item with `group`, `name`, `chargeMethod`, and `rate`.
_Avoid_: Bundle, record
